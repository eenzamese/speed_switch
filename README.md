## About The Project
[product-screen]

Enter needed values to only-gui legacy app with python

## Installation

Clone the repo
   ```sh
   git clone https://github.com/eenzamese/auto.git
   ```

## Usage

Edit inputs in auto.py
```py
# inputs
WINDOW_TITLE_RGX = ".*Window name.*"
INPUT = "some input string"
```

Run python script
   ```sh
   python3 auto/auto.py
   ```
